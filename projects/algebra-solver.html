Algebra Solver

Symbolic Algebra System in C++

Live Demo View Code

Demo

Overview

A computer algebra system that solves systems of linear equations and shows the step-by-step work. I built this to learn about parsing, expression trees, and symbolic computation.

Given a set of equations and a target variable, it substitutes known values, simplifies expressions, and isolates the variable to find the solution.

How It Works

  • Lexer

    Tokenizes input strings into numbers, variables, and operators.

  • Parser

    Builds an expression tree using Pratt parsing (operator precedence). Handles binary operations, unary minus, and parentheses.

  • Simplifier

    Reduces expressions by folding constants, combining like terms, and distributing multiplication.

  • Solver

    Uses a priority queue to process equations. Equations with fewer unknowns get solved first, then their values are substituted into remaining equations.

  • Isolator

    Rearranges an equation to get the target variable alone on one side.

Architecture

Core Engine

Written in C++17. AST nodes represent atoms (numbers, variables) and operations. The solver, simplifier, and isolator operate on these trees.

Python Bindings

pybind11 exposes the C++ functions to Python: simplify(), isolate(), solve(). This lets the web backend call the engine directly.

Backend

FastAPI server with endpoints for /simplify and /solve-system. Imports the compiled C++ module.

Frontend

React app where you enter equations and see the solution steps.

Limitations

Only handles linear equations. No quadratics, polynomials, or transcendental functions. Doesn't support inequalities.

Technology Stack
