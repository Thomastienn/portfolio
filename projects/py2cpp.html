Py2Cpp

Python to C++ Transpiler for Competitive Programming

Live Demo View Code

Demo

Overview

Py2Cpp converts Python code to C++ using AST-based parsing. I built it because I sometimes write solutions in Python first but need C++ for faster execution times on online judges. Instead of manually rewriting, this tool automates the translation.

It handles common Python patterns like loops, conditionals, list operations, and built-in functions. When the AST-based approach can't handle something, it falls back to an LLM.

How It Works

  • Parsing

    Python's ast module parses source code into an abstract syntax tree. The transpiler walks this tree and converts each node to equivalent C++ code.

  • Type Inference

    A linter pass infers variable types (int, string, vector, map, etc.) by analyzing assignments and function calls. This determines the C++ type declarations.

  • Built-in Translations

    Common Python functions get mapped to C++ equivalents: print() to cout, len() to .size(), range() to for loops, input() to a helper template, etc.

  • Template Helpers

    Some Python behaviors need helper functions in C++. These templates (for slicing, mapping, etc.) are only included when used.

  • LLM Fallback

    When the deterministic transpiler fails, Gemini API attempts the conversion. There's also an endpoint to fix compilation errors.

Architecture

Transpiler Core

ExprParser processes statements (assignments, loops, conditionals). ReprVisitor converts expressions to C++ strings. Linter tracks variable types and scopes.

Web Backend

FastAPI server with rate limiting (30 req/min), input size limits, and AST complexity checks to prevent abuse.

Frontend

React app with Monaco Editor for code input. Shows Python on the left, generated C++ on the right.

CLI

Can also run locally via command line: python main.py input.py

Limitations

Doesn't support classes, decorators, generators, or complex Python features. Type inference can fail on ambiguous cases. Designed for simple, procedural code.
